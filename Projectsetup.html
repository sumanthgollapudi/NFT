Dependencies and Tools:

Nodejs – used to connect project to the wallet and all other function. Like interacting with frontend using Web3Js.

Ganache – Personal blockchain, which gives sample accounts with some fake crypto. (Ganache is like real block chain, which run on our local system)

Truffle – Truffle Framework is used to develop Ethereum smart contract using solidity programming language. And this is used to build, migrate and deploy smart contract the network

Meta Mask- Browser extension to connect to Ethereum network or any other private network. Here we are importing sample accounts from ganache, to use in our project Basically this is used for authentication of an account and also to transfer data.
